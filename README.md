# Employee Payroll Management System

## Description
This project is a Python and SQLite based application that manages employee payroll records.
It calculates net salary using basic salary, allowance, and deductions.

## Technologies Used
- Python
- SQLite
- SQL (CRUD Operations)

## Features
- Add employee details
- View all employees
- Search employee by ID
- Update employee salary
- Delete employee records
- Automatic net salary calculation

## How to Run
1. Open terminal in project folder
2. Run:
   python main.py
3. Select options from menu
